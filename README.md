### 开始使用
1. **获取代码**: 访问我们的GitHub或Gitee仓库，克隆或下载最新的源代码。
2. **环境准备**: 确保你的开发环境中已安装Node.js和HBuilderX（推荐），这是uniapp的官方开发工具。
